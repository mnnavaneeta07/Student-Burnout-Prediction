# Student Burnout Prediction Using Digital Behavior

## Project Description
This project predicts the burnout level of students using machine learning techniques based on academic, behavioral, and lifestyle data.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Visual Studio Code

## Algorithm
- Decision Tree Classifier

## Features
- Dataset preprocessing
- Label encoding
- Model training
- Interactive user input
- Burnout level prediction
